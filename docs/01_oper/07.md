# Проведение документов
* удаление движений;
* блокировка;
* чтение данных;
* контроль остатков;
* запись движений.

## Приходная накладная
* только пишем, не читаем - блокировка не нужна;
* исключение дублей строк - группировка по номенклатуре;
* исключение услуг из складского учета;
* приходуем, поэтому контроль отрицательных остатков не нужен;
* вывод движений в типовой форме;

## Расходная накладная
* блокировка нужна:
    * [расчета себестоимости][1];
    * контроля отрицательных остатков;
* исключение услуг из складского учета;
* расходуем, поэтому контроль отрицательных остатков нужен;
* вывод движений в типовой форме;

[1] - не обязатеольно себестоимость, речь про расчет движений на основании данных регистров.
