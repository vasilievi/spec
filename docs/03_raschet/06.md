# Табель

Количество фактически отработанных часов вводится в систему с помощью документа «Табель». Для каждого сотрудника, на каждый день месяца, вводится количество фактически отработанных часов и больничный.


На форме в шапке документа поля «Номер», «Дата». Ниже табличная часть. В ней колонки «Сотрудник» и «1», «2», «3»...«31»
В колонке «Сотрудник» отображается список сотрудников, а на пересечении сотрудника и дня месяца один из трех вариантов: «8», «Б» либо пустое поле.
